Sensitive configuration paths reference

This topic lists only configuration paths for sensitive settings. For a list of other configuration paths, see:

  • All configuration paths except payments
  • Payment configuration paths.

To use a configuration path as a variable:

  • All text must be ALL CAPS.
  • Prefix the configuration path with the scope (the default scope, CONFIG__DEFAULT, or a specific scope).
  • Replace / characters in the configuration path with two underscore characters.

All configuration paths listed in this topic are sensitive. The System-specific? column shows which values are also system-specific.
