Component Status

Last Updated: June 23, 2017

This page shows the ongoing status of each component during the Magento 2.2 Release Candidate period and is intended to show you the readiness of different components.

Each status color indicates the severity and amount of 2.2-related issues for a particular component.

  • Red - The module has critical and blocking issues that make this module unstable.
  • Yellow - The module has some critical and major issues that may cause some problems.
  • Green - The module has a few issues, but major functionality is not affected.

The issues used to generate the status of these components come from verified internal and GitHub reports for Magento 2.2 that are planned to be fixed prior to the General Availability release. Issues that are not related to the 2.2 release are not part of this report.
