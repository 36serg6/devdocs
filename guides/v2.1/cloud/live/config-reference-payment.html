Payment configuration paths reference

This topic lists payment-related configuration paths. For a list of other configuration paths, see:

  • All configuration paths except payments
  • Sensitive configuration paths

To optionally override system-specific settings, you can use configuration variables.

To use a configuration path as a variable:

  • All text must be ALL CAPS.
  • Prefix the configuration path with the scope (the default scope, CONFIG__DEFAULT, or a specific scope).
  • Replace / characters in the configuration path with two underscore characters.

For more information, see How to name configuration variables.

Payment methods paths

These configuration values are availabe in the Magento Admin in Stores > Settings > Configuration > Sales > Payment Methods.

The settings are further organized by payment method.
